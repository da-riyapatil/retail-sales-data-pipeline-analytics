# Power BI Service Features Used

This project was also published and managed using Power BI Service, where the following features were explored:

## Features Used

- **Dashboard Publishing**
  - The report was published from Power BI Desktop to Power BI Service for online access.

- **Scheduled Refresh**
  - Data refresh was configured to keep the dashboard up to date with the latest data.

- **Row-Level Security (RLS)**
  - Basic RLS concepts were explored to restrict data visibility for different users.

- **Gateway Setup**
  - A gateway was used to connect local data sources with Power BI Service.

- **Dataflows**
  - Dataflows were used to manage and prepare data within Power BI Service.

- **Mobile View**
  - The dashboard was optimized for mobile viewing.

- **Data Alerts**
  - Alerts were configured on key metrics to track important changes.

---

## Note

These features were explored at a practical level as part of understanding how dashboards are managed and deployed in real-world scenarios.
